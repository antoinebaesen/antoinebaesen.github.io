    Types de base en informatique

    Introduction

    On a vu qu'on pouvait représenter nombres et caractères mais il y a d'autres types de base en informatique. On va voir ici les types de base en Python, le langage de programmation qu'on utilise en NSI.

    Les types de base en Python

    On a déjà vu trois types de base en Python :

    • Entiers : Les entiers sont des nombres entiers positifs ou négatifs. En Python, on peut les écrire sans limite de taille. Par exemple, 42 est un entier.
    • Flottants : Les flottants sont des nombres à virgule. En Python, on utilise le point pour séparer la partie entière de la partie décimale. Par exemple, 3.14 est un flottant.
    • Chaines de caractères : Les chaines de caractères sont des suites de caractères. En Python, on les écrit entre guillemets simples ou doubles. Par exemple, "Hello, World!" est une chaine de caractères.

    En plus de ces types qu'on a déjà vus, il y a d'autres types de base en Python :

    Booléens

    Les booléens sont des valeurs qui peuvent être soit True (vrai) soit False (faux). En Python, on utilise les mots-clés True et False pour représenter ces valeurs. Par exemple, True est un booléen.

    On peut faire des opérations logiques sur les booléens.

    Opérations logiques

    • and : L'opération and est un ET logique. Elle renvoie True si les deux opérandes sont True, sinon elle renvoie False.
    • or : L'opération or est un OU logique. Elle renvoie True si au moins un des opérandes est True, sinon elle renvoie False.
    • not : L'opération not est un NON logique. Elle renvoie True si l'opérande est False, et False si l'opérande est True.- xor : L'opération xor est un OU exclusif. Elle renvoie True si exactement un des opérandes est True, sinon elle renvoie False.

    Pour résumer les opérations logiques on va utiliser cette table de vérité :

    a b a and b a or b not a a xor b
    True True True True False False
    True False False True False True
    False True False True True True
    False False False False True False

    On peut aussi utiliser des parenthèses pour changer l'ordre d'évaluation des opérations logiques. Exemple : (a and b) or c, avec a = True, b = False et c = True (True and False) or True est équivalent à False or True qui est équivalent à True.

    On peut aussi faire des comparaisons entre des valeurs pour obtenir des booléens.

    Exercice 1

    Evaluer les expressions suivantes avec a = True, b = False et c = True.

    1. a and b
    2. a or b
    3. not a or c
    4. (a and b) xor (c or a)

    Comparaisons

    On peut aussi comparer des valeurs pour obtenir des booléens. Pour cela on peut utiliser les opérateurs de comparaison suivants :

    • == : L'opérateur == permet de tester l'égalité entre deux valeurs. Il renvoie True si les deux valeurs sont égales, sinon il renvoie False.
    • != : L'opérateur != permet de tester la différence entre deux valeurs. Il renvoie True si les deux valeurs sont différentes, sinon il renvoie False.
    • < : L'opérateur < permet de tester si une valeur est inférieure à une autre. Il renvoie True si la première valeur est inférieure à la deuxième, sinon il renvoie False.
    • <= : L'opérateur <= permet de tester si une valeur est inférieure ou égale à une autre. Il renvoie True si la première valeur est inférieure ou égale à la deuxième, sinon il renvoie False.
    • > : L'opérateur > permet de tester si une valeur est supérieure à une autre. Il renvoie True si la première valeur est supérieure à la deuxième, sinon il renvoie False.
    • >= : L'opérateur >= permet de tester si une valeur est supérieure ou égale à une autre. Il renvoie True si la première valeur est supérieure ou égale à la deuxième, sinon il renvoie False.

    Exercice 2

    Evaluer les expressions suivantes avec a = 42, b = 3.14 et c = 40.

    1. a == b
    2. a != c
    3. (a < c) and (b > c)
    4. (a - b) >= c
    5. (a < 50) xor (c > 20)

    None

    None est une valeur spéciale en Python qui représente l'absence de valeur. On l'utilise pour initialiser des variables qui n'ont pas encore de valeur. Par exemple, None est une valeur.

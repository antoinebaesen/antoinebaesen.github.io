    Base de langage Python

    Introduction

    Au cours de nos cours de NSI, nous allons utiliser le langage de programmation python pour illustrer les concepts abordés.

    Concepts de base

    Programme

    Un programme est une suite d'instructions qui permettent de réaliser une tâche. En python, un programme est une suite d'instructions écrites dans un fichier avec l'extension .py.

    Pour travailler sur nos programmes, nous allons utiliser un éditeur de texte appelé Thonny.

    Dans Thonny on peut ouvrir un programme avec le menu Fichier puis Ouvrir ou en cliquant sur l'icône Ouvrir dans la barre d'outils.

    Pour exécuter un programme, on peut cliquer sur le bouton Exécuter dans la barre d'outils ou utiliser le raccourci clavier F5.

    Variables

    Une variable est un espace de stockage en mémoire qui contient une valeur et qui est définie par un nom. En python, on peut déclarer une variable en lui affectant une valeur.

    Les variables peuvent contenir différents types de données comme on les a vus précédemment. (Entier, flottant, chaîne de caractères, booléen et d'autres types de données que nous verrons plus tard).

    unNombre = 5
unTexte = "Bonjour"
unFlottant = 3.14
unBooleen = True

    Les Variables, comme leur nom l'indique, peuvent varier. On peut donc modifier la valeur d'une variable en lui affectant une nouvelle valeur.

    # Ici on déclare une variable unNombre et on lui affecte la valeur 5
unNombre = 5

# Ici on modifie la valeur de la variable unNombre
unNombre = 10

    Les commentaires en python commencent par le symbole #. Toute la ligne après le symbole # est un commentaire et n'est pas exécutée.

    La fonction print() permet d'afficher du texte ou des valeurs à l'écran.

    print("Bonjour")
# Affiche Bonjour à l'écran

    Opérations arithmétiques

    Python permet de réaliser des opérations arithmétiques sur des variables numériques.

    a = 5
b = 3

# Addition
resultat = a + b

# Soustraction
resultat = a - b

# Multiplication
resultat = a * b

# Division
resultat = a / b

# Division entière
resultat = a // b

# Reste de la division
resultat = a % b

# Puissance
resultat = a ** b

    Opérations sur les chaînes de caractères

    Concaténation

    Python permet de réaliser des opérations sur les chaînes de caractères.

    
# Concaténation : On peut coller deux chaînes de caractères ensemble pour en former une seule avec l'opérateur +

a = "Bonjour"
b = "tout le monde"

resultat = a + " " + b
print(resultat)

    Longueur d'une chaîne

    Python permet de connaître la longueur d'une chaîne de caractères avec la fonction len().

    a = "Bonjour"
longueur = len(a)
print("La chaîne a une longueur de", longueur, "caractères")

    Fonctions

    Une fonction est un bloc d'instructions qui réalise une tâche spécifique. En python, on peut définir une fonction avec le mot-clé def.

    def maFonction():
    print("Bonjour")

    Les fonctions peuvent prendre des paramètres.

    def maFonction(nom):
    print("Bonjour", nom)

    Les fonctions sont écrite avec le format suivant :

    def nomDeLaFonction(parametre1 : typeDuParamètre, parametre2 : typeDuParamètre2) -> typeDuRetour(si la fonction retourne une valeur) :
    # Instructions

    Types de paramètres

    Pour savoir comment écrire les types des paramètres, on peut se référer à la table suivante :

    Type de données Type en python
    Entier int
    Flottant float
    Chaine de caractères str
    Booléen bool

    Par exemple :

    def majuscule(texte : str) -> str:
    return texte.upper()

def double(nombre : int) -> int:
    return nombre * 2

def faisPouet():
    print("Pouet")

    Pour appeler une fonction, on utilise son nom suivi de parenthèses.

    maFonction(parametre1, parametre2 ...)

    Une fontion peut prendre entre 0 et n paramètres.

    Une fonction peut retourner une valeur avec le mot-clé return.

    def donneMoi3():
    return 3

monNombre = donneMoi3()
# monNombre contient la valeur 3
print(monNombre)

    Structures conditionnelles

    Les structures conditionnelles permettent d'exécuter des instructions en fonction d'une condition.

    on l'écrit sous la forme :

    if condition:
    # Instructions
elif autreCondition:
    # Instructions
else:
    # Instructions

    elif est une contraction de else if. On peut en mettre autant que l'on veut. else et elif sont optionnels, on peut ne mettre que if.

    Par exemple :

    nombre = 5

if nombre >= 0:
    print("Le nombre est positif")
else :
    print("Le nombre est négatif")

    Boucles bornées

    Plutôt que de répéter plusieurs fois la même instruction, on peut utiliser une boucle pour répéter une instruction ou un bloc d'instructions.

    Par exemple si on veut afficher 5 fois "Bonjour" :

    print("Bonjour")
print("Bonjour")
print("Bonjour")
print("Bonjour")
print("Bonjour")

    C'est une écriture fastidieuse. On peut utiliser une boucle for pour répéter l'instruction.

    for i in range(5):
    print("Bonjour")

    C'est beaucoup plus court et plus lisible.

    Décortiquons la boucle for :

    • for est le mot-clé qui indique que l'on va faire une boucle.
    • i est une variable qui va prendre les valeurs de range(5). On peut mettre n'importe quel nom de variable.
    • in est le mot-clé qui indique que l'on va parcourir les éléments d'une liste.
    • range(5) est une fonction qui génère une liste de 5 éléments. range(5) génère la liste [0, 1, 2, 3, 4].
    • : indique que l'on va commencer un bloc d'instructions. La suite du programme doit être indentée. (Décalée de 4 espaces ou d'une tabulation).
    • Tout ce qui est à la suite et indenté va être répété par la boucle.

    Autre exemple

    On peut aussi parcourir une liste.

    # La liste est définie entre crochets [], c'est une serie de valeurs séparées par des virgules.
liste = [2, 7, 4, 5]

for element in liste:
    print(element)

    Boucles non bornées

    Il existe une autre boucle en python, la boucle while. Cette boucle répète une instruction ou un bloc d'instructions tant qu'une condition est vraie.

    Elle est écrite au format suivant :

    while CONDITION:
    # Instructions

    Par exemple :

    i = 0

while i < 5:
    print("Bonjour")
    i = i + 1

    Ici, on incrémente la variable i à chaque tour de boucle pour qu'à un moment donné la condition i < 5 soit fausse et que la boucle s'arrête. C'est l'équivalent de la boucle for que l'on a vu précédemment.
